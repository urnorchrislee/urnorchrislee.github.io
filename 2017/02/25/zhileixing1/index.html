值类型和引用类型（一）简单说明

发表于 2017-02-25 | 分类于 数据类型
  • 值类型
    • string / number / boolean / null / undefined
  • 引用类型
    • Object / Function / Array / Date / String / Number / Boolean

区别:在于空间中存储的信息
值类型:存储的是具体的数据
引用类型:存储的是一个引用(reference)[地址],该引用指向内存中的一块儿空间，这块儿空间存储着具体的数据
举例：
01(变量的名称) –> 一个苹果(具体的数据)
02(变量的名称) –> 苹果在超市的第二排货架的第一列(地址)
