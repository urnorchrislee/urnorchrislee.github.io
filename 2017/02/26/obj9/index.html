面向对象(九)扩展内置对象

发表于 2017-02-26 | 分类于 面向对象

扩展内置对象
需求：在数组上添加一个属性和方法
实现：利用对象的动态特性添加成员（属性/方法）

方式一 直接在原型上扩展内置对象

注意：在扩展内置构造函数的时候不要使用上面的方法
是因为原型对象上面的属性和方法会被所有的对象共享，可能会出现一莫名其妙的问题
如果直接在原型对象上添加属性和方法，那么很可能会被覆盖

01 提供一个构造函数（自定义）
02 设置构造函数的原型对象为内置构造函数创建出来的对象
