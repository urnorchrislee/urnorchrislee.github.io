面向对象(十)深拷贝和浅拷贝

发表于 2017-02-26 | 分类于 面向对象

深拷贝

  • 提供一个函数，两个参数（元对象，要拷贝属性的对象）
  • 在函数中先检查第一个参数是否有值，如果没有值那么就初始化一个空的对象
  • for..in循环来遍历参数2,
    • 检查当前的属性值是什么类型
    • 如果是值类型，那么就直接拷贝赋值
    • 如果是引用类型，那么就再调用一次这个方法，去内部拷贝这个对象的所有属性
